This is a package to analyze cds-bond basis

CDS Prices will be pulled from Databricks

Bond prices will be pulled from the bloomberg desktop api, with the base field being "BLOOMBERG_MID_G_SPREAD"

We aim to compute both the spread and dollar price difference between bonds and cds over time

Our base will will be using the "Best_Bond_Proxy_ISIN" which is found in the cds_mappings ("S:\Structured Credit\Matt Stuff\cds_mappings")
